How to use?

Adding projects

You can add new project by clicking on the adding project button in the Adding spot.

Adding tasks

Write the task you want to do after clicking on adding task button.

Tracking

Set the date when you should finish your task , otherwise the uncompleted task will be moved to the uncompleted tasks section.

Editing

Edit your task if needed by clicking on the pen icon on each task separately.
